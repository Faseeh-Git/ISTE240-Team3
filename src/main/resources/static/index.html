Campus Event Organizer Platform

This platform allows university clubs and organizers to create and manage events while students can discover, register, and track campus activities.

Features

  • Student and Organizer Registration & Login
  • User Profile Management
  • Create, Edit and Delete Events
  • Event Listing and Detailed Event Pages
  • Student Event Registration and Booking System
  • Event Filtering by Category and Date
  • Administrator Panel for Managing Users and Events
  • Statistical Dashboard (Total Events, Registrations, Popular Categories)
